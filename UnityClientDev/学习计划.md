# Unity 客户端 & 游戏服务端开发双轨学习计划

> **目标：** 系统掌握游戏开发全栈能力，从 C# 客户端开发进阶到高性能游戏服务端开发
> **方法：** 理论 + 动手交替进行，每个知识点都有可运行的示例
> **前置：** 有 C++（Raylib 13 个项目）和 Lua（Pico-8）基础

## 总览

| 阶段 | 内容 | 对标 Raylib/Pico-8 |
|------|------|-------------------|
| 基础 | 01 C# 语法与 OOP | C++ 类、Lua 表 |
| 核心 | 02 Unity 引擎核心 | 帧循环、组件、输入 |
| 进阶 | 03 Unity 进阶主题 | 协程、资源管理、渲染 |
| 综合 | 04 客户端专业技能 | 网络、AI、序列化、优化 |
| 实战 | 05 小游戏实战 | Breakout、Pong 的 Unity 版 |
| 设计 | 06 设计模式 | 单例、对象池、状态机 |
| 工具 | 07 辅助工具与 SDK | 原生交互、打包脚本、SDK接入 |
| 服务 | 08 游戏服务端开发 | Linux、Epoll、Redis、GM工具 |
| 面试 | 09 面试冲刺 | 已有题库系统复习 |

## 学习模式说明

> 每个主题有 **基础版**（`DayXX_主题.md`）和 **深入版**（`DayXX_主题_深入.md`）两个文件。

| 文件类型 | 说明 | 建议时间 |
|---------|------|---------|
| 基础版 | 核心概念 + 基础用法 + 简单示例 | 20-30 分钟 |
| 深入版 | 进阶原理 + 底层机制 + 实战技巧 | 30-45 分钟 |

**推荐学习路径：**
1. **快速入门**：只读基础版 → 完成练习 → 进入下一主题
2. **深入理解**：读完基础版 → 读深入版 → 对比 C++/Raylib 差异
3. **查漏补缺**：遇到具体问题时，针对性阅读深入版相关章节

## 时间安排

> 你有 Unity 基础 + C++/Lua 经验，重点放在 **C# 特有的概念**、**自动化工具链** 以及 **高并发服务端体系**。

```
22:00 ─ 22:05  回顾昨天的笔记（5分钟热身）
22:05 ─ 22:25  读概念 + 看示例代码（20分钟）
22:25 ─ 22:40  动手改代码 / 写小练习（15分钟）
22:40 ─ 22:45  记录一条你今天学到的东西（停靠点）
22:45         停！关电脑，让大脑离线
```
