### 1.1 面向对象编程三大特性

#### Q1: 什么是面向对象编程？它相比面向过程编程有什么优势？

**参考答案**：

面向对象编程（OOP）是一种程序设计范式，将程序视为由相互协作的对象集合，每个对象包含数据和行为。

**三大特性**：

| 特性 | 说明 | 优势 |
|-------|------|------|
| 封装 | 把数据和操作封装到对象中，对外隐藏实现细节 | 提高安全性、降低耦合度 |
| 继承 | 子类继承父类的属性和方法 | 代码复用、减少重复 |
| 多态 | 同一接口不同实现 | 灵活性、可扩展性 |

**对比**：

| 面向过程 | 面向对象 |
|---------|---------|
| 数据和函数分离 | 数据和行为封装在一起 |
| 解决问题步骤 | 解决问题实体 |
| 难以复用 | 继承复用 |

#### Q2: 详细解释 C# 中的封装是如何实现的？

**参考答案**：

C# 通过访问修饰符实现封装：

```csharp
public class Player
{
    // 私有字段 - 外部无法直接访问
    private int _health;
    private string _name;
    
    // 属性 - 提供受控的访问
    public int Health 
    { 
        get => _health; 
        set => _health = value >= 0 ? value : 0;
    }
    
    // 公开方法
    public void TakeDamage(int damage)
    {
        Health -= damage;
        Console.WriteLine($"受到 {damage} 点伤害，剩余 {Health} HP");
    }
}
```

**访问修饰符级别**：

| 修饰符 | 访问范围 |
|--------|---------|
| public | 任何地方 |
| private | 当前类内部 |
| protected | 当前类及子类 |
| internal | 当前程序集 |
| protected internal | 当前程序集或子类 |
| private protected | 当前类及其子类（同程序集） |

### 1.2 C# 垃圾回收机制

#### Q5: 什么是 GC？C# 中的垃圾回收机制是如何工作的？

**参考答案**：

GC（Garbage Collection）自动回收不再使用的内存。

**托管堆vs非托管资源**：

| 类型 | 管理方式 | 例子 |
|------|---------|------|
| 托管对象 | GC 自动回收 | 对象、数组 |
| 非托管资源 | 手动释放 | 文件、数据库连接 |

**GC 工作原理**：

```csharp
class Program
{
    static void Main()
    {
        // 创建对象 - 在托管堆分配
        Player p = new Player();  // 引用在栈，数据在堆
        
        p = null;  // 断开引用
        
        // 现在 p 指向的对象成为"垃圾"
        // GC 下次运行时将回收这块内存
    }
}
```

**GC 代际系统**：

| 代 | 对象存活时间 | 频率 |
|-----|-------------|------|
| Gen 0 | 短时间 | 最频繁 |
| Gen 1 | 中等时间 | 较少 |
| Gen 2 | 长时间 | 很少 |

#### Q6: 如何正确管理非托管资源？请解释 Dispose 模式

**参考答案**：

```csharp
public class GameResource : IDisposable
{
    private bool _disposed = false;
    private IntPtr _handle;  // 非托管资源
    
    // 模拟非托管资源
    public GameResource()
    {
        _handle = Marshal.AllocHGlobal(1024);  // 分配内存
    }
    
    public void Use()
    {
        CheckDisposed();
        Console.WriteLine("使用资源中...");
    }
    
    // 实现Dispose接口
    public void Dispose()
    {
        Dispose(true);
        GC.SuppressFinalize(this);  // 阻止finalizer
    }
    
    protected virtual void Dispose(bool disposing)
    {
        if (!_disposed)
        {
            if (_handle != IntPtr.Zero)
            {
                Marshal.FreeHGlobal(_handle);  // 释放内存
                _handle = IntPtr.Zero;
            }
            _disposed = true;
        }
    }
    
    // Finalizer - 防止忘记调用Dispose
    ~GameResource()
    {
        Dispose(false);
    }
}

// 使用方式 - using 语句自动释放
class Program
{
    static void Main()
    {
        using (var resource = new GameResource())
        {
            resource.Use();
        }  // 自动调用 Dispose
        
        // 等价于：
        // try { resource.Use(); }
        // finally { resource.Dispose(); }
    }
}
```

**最佳实践**：
1. 实现 `IDisposable` 接口处理非托管资源
2. 使用 `using` 语句确保释放
3. 在 Dispose 中调用 `GC.SuppressFinalize()`

### 1.3 装箱与拆箱

#### Q7: 什么是装箱和拆箱？请举例说明

**参考答案**：

```csharp
class Program
{
    static void Main()
    {
        // 装箱：值类型 → 引用类型
        int value = 100;          // 在栈上
        object boxed = value;       // 装箱，拷贝到堆上
        
        // 拆箱：引用类型 → 值类型
        int unboxed = (int)boxed;  // 拆箱，从堆拷贝到栈
        
        // 性能影响
        for (int i = 0; i < 100000; i++)
        {
            object o = i;        // 每次循环都装箱！性能损耗
        }
    }
}
```

**图示**：

```
装箱：int(栈) → 堆上的Box对象
┌─────────┐     ┌─────────────┐
│   100   │ ──→ │ Object:100  │
│ (栈)    │     │   (堆)      │
└─────────┘     └─────────────┘

拆箱：堆上的Box → int(栈)
┌─────────────┐     ┌─────────┐
│ Object:100 │ ──→ │   100   │
│   (堆)      │     │ (栈)    │
└─────────────┘     └─────────┘
```

**性能影响**：

| 操作 | 性能损耗 |
|------|---------|
| 装箱 | 中（内存分配+拷贝） |
| 拆箱 | 低（只需类型检查+拷贝） |
| 大量循环装箱 | 高 |

```csharp
// 栈 - 撤销系统
public class UndoManager
{
    private Stack<IAction> _undoStack = new Stack<IAction>();
    private Stack<IAction> _redoStack = new Stack<IAction>();
    
    public void Execute(IAction action)
    {
        action.Do();
        _undoStack.Push(action);
        _redoStack.Clear();  // 清除重做
    }
    
    public void Undo()
    {
        if (_undoStack.Count > 0)
        {
            var action = _undoStack.Pop();
            action.Undo();
            _redoStack.Push(action);
        }
    }
}

    }

    {
        return _queue.Dequeue();  // 先加入的先执行
    }
}
```

### 2.3 排序算法

#### Q3: 常见的排序算法有哪些？时间复杂度是多少？

**参考答案**：

| 算法 | 最好 | 平均 | 最坏 | 稳定性 | 适用场景 |
|------|------|------|------|--------|---------|
| 插入排序 | O(n) | O(n²) | O(n²) | 稳定 | 小规模/基本有序 |
| 希尔排序 | O(n) | O(n^1.3) | O(n²) | 不稳定 | 中等规模 |
| 快速排序 | O(nlogn) | O(nlogn) | O(n²) | 不稳定 | 大规模通用 |
| 归并排序 | O(nlogn) | O(nlogn) | O(nlogn) | 稳定 | 外部排序 |
| 堆排序 | O(nlogn) | O(nlogn) | O(nlogn) | 不稳定 | 优先队列 |

```csharp
// 快速排序
public void QuickSort(int[] arr, int low, int high)
{
    if (low < high)
    {
        int pivot = Partition(arr, low, high);
        QuickSort(arr, low, pivot - 1);
        QuickSort(arr, pivot + 1, high);
    }
}

private int Partition(int[] arr, int low, int high)
{
    int pivot = arr[high];
    int i = low - 1;
    
    for (int j = low; j < high; j++)
    {
        if (arr[j] <= pivot)
        {
            i++;
            (arr[i], arr[j]) = (arr[j], arr[i]);
        }
    }
    (arr[i + 1], arr[high]) = (arr[high], arr[i + 1]);
    return i + 1;
}
```

### 2.4 搜索算法

#### Q4: 二分查找适用于什么场景？请手写实现

**参考答案**：

```csharp
// 二分查找 - 必须是有序数组
public int BinarySearch<T>(T[] arr, T target) where T : IComparable
{
    int left = 0;
    int right = arr.Length - 1;
    
    while (left <= right)
    {
        int mid = left + (right - left) / 2;
        int cmp = arr[mid].CompareTo(target);
        
        if (cmp == 0)
            return mid;  // 找到
        else if (cmp < 0)
            left = mid + 1;
        else
            right = mid - 1;
    }
    
    return -1;  // 未找到
}

// 时间复杂度：O(logn) - 每次排除一半
```

**游戏中的应用**：
- 玩家排名查找
- 有序装备属性查找
- 任务ID查找

### 2.5 递归与动态规划

#### Q5: 什么是动态规划？请举例说明

**参考答案**：

动态规划适用于有重复子问题的情况，通过保存子问题结果避免重复计算。

```csharp
// 斐波那契 - 递归版本（效率低）
public int FibRecursive(int n)
{
    if (n <= 1) return n;
    return FibRecursive(n - 1) + FibRecursive(n - 2);
    // 时间复杂度 O(2^n) - 指数爆炸
}

// 动态规划版本（高效）
public int FibDP(int n)
{
    if (n <= 1) return n;
    
    int[] dp = new int[n + 1];
    dp[0] = 0;
    dp[1] = 1;
    
    for (int i = 2; i <= n; i++)
    {
        dp[i] = dp[i - 1] + dp[i - 2];
    }
    
    return dp[n];
    // 时间复杂度 O(n)
}
```

**背包问题**：

```csharp
// 0-1背包 - 动态规划
public int Knapsack(int[] values, int[] weights, int capacity)
{
    int n = values.Length;
    int[,] dp = new int[n + 1, capacity + 1];
    
    for (int i = 1; i <= n; i++)
    {
        for (int w = 0; w <= capacity; w++)
        {
            if (weights[i-1] <= w)
            {
                dp[i, w] = Math.Max(
                    dp[i-1, w],
                    dp[i-1, w-weights[i-1]] + values[i-1]
                );
            }
            else
            {
                dp[i, w] = dp[i-1, w];
            }
        }
    }
    
    return dp[n, capacity];
}
```

### 2.6 A* 算法

#### Q6: 简述 A* 寻路算法原理

**参考答案**：

A* 是游戏中最常用的寻路算法，综合了 Dijkstra 和贪心。

```csharp
public class AStar
{
    // 估价函数：F = G + H
    // G：从起点到当前点实际代价
    // H：从当前点到终点估计代价（启发式）
    
    public List<Point> FindPath(Map map, Point start, Point end)
    {
        var openSet = new PriorityQueue<Node, int>();
        var closedSet = new HashSet<Point>();
        var cameFrom = new Dictionary<Point, Point>();
        
        openSet.Enqueue(new Node(start, 0, Heuristic(start, end)), 0);
        
        while (openSet.Count > 0)
        {
            var current = openSet.Dequeue();
            
            if (current.Point == end)
                return ReconstructPath(cameFrom, start, end);
            
            closedSet.Add(current.Point);
            
            foreach (var neighbor in GetNeighbors(current.Point))
            {
                if (closedSet.Contains(neighbor)) continue;
                
                var g = current.G + 1;
                var h = Heuristic(neighbor, end);
                openSet.Enqueue(new Node(neighbor, g, h), g + h);
                cameFrom[neighbor] = current.Point;
            }
        }
        
        return null;  // 无路径
    }
    
    // 曼哈顿距离 - 启发式函数
    private int Heuristic(Point a, Point b)
    {
        return Math.Abs(a.X - b.X) + Math.Abs(a.Y - b.Y);
    }
}
```

**估价函数选择**：

| 启发式 | 公式 | 适用场景 |
|--------|------|---------|
| 曼哈顿 | |x1-x2| + |y1-y2| | 4方向移动 |
| 对角 | max(|dx|, |dy|) + (√2-2)*min(|dx|,|dy|) | 8方向移动 |
| 欧几里得 | √((x1-x2)² + (y1-y2)²) | 任意方向 |

```csharp
                return NodeStatus.Success;
            if (status == NodeStatus.Running)
                return NodeStatus.Running;
```

```csharp
public class QuickSort
{
    public void Sort(int[] arr, int left, int right)
    {
        if (left >= right) return;
        
        int pivot = Partition(arr, left, right);
        Sort(arr, left, pivot - 1);
        Sort(arr, pivot + 1, right);
    }
    
    private int Partition(int[] arr, int left, int right)
    {
        int pivot = arr[right];  // 选择最右边的作为基准
        int i = left - 1;
        
        for (int j = left; j < right; j++)
        {
            if (arr[j] <= pivot)
            {
                i++;
                (arr[i], arr[j]) = (arr[j], arr[i]);  // 交换
            }
        }
        (arr[i + 1], arr[right]) = (arr[right], arr[i + 1]);
        return i + 1;
    }
}

// 测试
```

```csharp
public class MergeSort
{
    private int[] _temp;
    
    public void Sort(int[] arr)
    {
        _temp = new int[arr.Length];
        Sort(arr, 0, arr.Length - 1);
    }
    
    private void Sort(int[] arr, int left, int right)
    {
        if (left >= right) return;
        
        int mid = left + (right - left) / 2;
        Sort(arr, left, mid);
        Sort(arr, mid + 1, right);
        Merge(arr, left, mid, right);
    }
    
    private void Merge(int[] arr, int left, int mid, int right)
    {
        // 复制到临时数组
        for (int i = left; i <= right; i++)
            _temp[i] = arr[i];
        
        int i = left, j = mid + 1, k = left;
        
        while (i <= mid && j <= right)
        {
            if (_temp[i] <= _temp[j])
                arr[k++] = _temp[i++];
            else
                arr[k++] = _temp[j++];
        }
        
        while (i <= mid)
            arr[k++] = _temp[i++];
    }
}
```

```csharp
public class HeapSort
{
    public void Sort(int[] arr)
    {
        int n = arr.Length;
        
        // 构建最大堆
        for (int i = n / 2 - 1; i >= 0; i--)
            Heapify(arr, n, i);
        
        // 堆排序
        for (int i = n - 1; i > 0; i--)
        {
            (arr[0], arr[i]) = (arr[i], arr[0]);  // 交换
            Heapify(arr, i, 0);
        }
    }
    
    private void Heapify(int[] arr, int n, int i)
    {
        int largest = i;
        int left = 2 * i + 1;
        int right = 2 * i + 2;
        
        if (left < n && arr[left] > arr[largest])
            largest = left;
        if (right < n && arr[right] > arr[largest])
            largest = right;
        
        if (largest != i)
        {
            (arr[i], arr[largest]) = (arr[largest], arr[i]);
            Heapify(arr, n, largest);
        }
    }
}
```

### 9.3 手写A*算法核心

```csharp
// 简化的A*算法实现
public class SimpleAStar
{
    // 估价函数：f = g + h
    public static int F(int g, int h) => g + h;
    
    // 曼哈顿距离
    public static int Heuristic(int x1, int y1, int x2, int y2)
    {
        return Math.Abs(x1 - x2) + Math.Abs(y1 - y2);
    }
    
    // 核心逻辑（伪代码）
    public List<(int, int)> FindPath(int startX, int startY, int endX, int endY)
    {
        // 1. 将起点加入开放列表
        var openSet = new PriorityQueue<(int, int), int>();
        var closedSet = new HashSet<(int, int)>();
        var cameFrom = new Dictionary<(int, int), (int, int)>();
        var gScore = new Dictionary<(int, int), int>();
        
        gScore[(startX, startY)] = 0;
        openSet.Enqueue((startX, startY), Heuristic(startX, startY, endX, endY));
        
        while (openSet.Count > 0)
        {
            var current = openSet.Dequeue();
            
            // 到达终点
            if (current == (endX, endY))
                return ReconstructPath(cameFrom, current);
            
            closedSet.Add(current);
            
            // 遍历邻居（上下左右）
            var neighbors = new[] { (0, 1), (0, -1), (1, 0), (-1, 0) };
            foreach (var (dx, dy) in neighbors)
            {
                var next = (current.Item1 + dx, current.Item2 + dy);
                if (closedSet.Contains(next)) continue;
                
                var tentativeG = gScore[current] + 1;
                
                if (!gScore.ContainsKey(next) || tentativeG < gScore[next])
                {
                    cameFrom[next] = current;
                    gScore[next] = tentativeG;
                    var f = tentativeG + Heuristic(next.Item1, next.Item2, endX, endY);
                    openSet.Enqueue(next, f);
                }
            }
        }
        
        return null;  // 无路径
    }
}
```
