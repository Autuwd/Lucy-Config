# 一、Unity 编辑器（高频操作界面）

## Tier 1 — 每天必见 ★★★

| # | 单词 | 音标 | 中文 | 详解 & 例句 |
|---|------|------|------|-------------|
| 1 | **Scene** | /siːn/ | 场景视图 | 3D/2D 可视编辑区。你在 Scene 视图中摆放、移动、旋转游戏对象。<br>📌 *"Drag the cube into the Scene."* |
| 2 | **Game** | /ɡeɪm/ | 游戏视图 | 模拟玩家看到的最终画面（摄像机输出）。<br>📌 *"Press Play to test in the Game view."* |
| 3 | **Hierarchy** | /ˈhaɪərɑːrki/ | 层级面板 | 当前场景中所有对象的树形列表。父子关系在此管理。<br>📌 *"Find the Player object in the Hierarchy."* |
| 4 | **Inspector** | /ɪnˈspektər/ | 检视面板 | 选中对象后显示其所有组件和属性，值都在这里改。<br>📌 *"Adjust the speed in the Inspector."* |
| 5 | **Project** | /ˈprɒdʒekt/ | 项目面板 | 显示项目文件夹中所有资源文件（模型、贴图、脚本等）。<br>📌 *"Import the texture into the Project window."* |
| 6 | **Console** | /kənˈsəʊl/ | 控制台 | 显示日志、警告、错误信息。调试必备。<br>📌 *"Check the Console for errors."* |
| 7 | **Play** | /pleɪ/ | 播放/运行 | ▶ 按钮，进入运行模式测试游戏。<br>📌 *"Click Play to test your changes."* |
| 8 | **Pause** | /pɔːz/ | 暂停 | ⏸ 暂停运行模式，便于分析当前帧。<br>📌 *"Pause the game to inspect this frame."* |
| 9 | **Step** | /step/ | 逐帧执行 | ▶▶ 每点击一次前进一帧，用于精细调试。<br>📌 *"Use Step to advance frame by frame."* |

## Tier 2 — 常用操作 ★★★

| # | 单词 | 音标 | 中文 | 详解 & 例句 |
|---|------|------|------|-------------|
| 10 | **GameObject** | /ɡeɪm əbˈdʒekt/ | 游戏对象 | 场景中的一切实体。任何物体都是 GameObject + 组件。<br>📌 *"Every character is a GameObject."* |
| 11 | **Component** | /kəmˈpəʊnənt/ | 组件 | 附加在 GameObject 上的功能模块（Rigidbody、Collider 等）。<br>📌 *"Add a Rigidbody component for physics."* |
| 12 | **Transform** | /trænsˈfɔːrm/ | 变换组件 | 控制对象的位置(Position)、旋转(Rotation)、缩放(Scale)。<br>📌 *"Reset the Transform to zero."* |
| 13 | **Position** | /pəˈzɪʃn/ | 位置 | Transform 中的 X/Y/Z 坐标值。<br>📌 *"Set Position Y to 5 to raise the object."* |
| 14 | **Rotation** | /rəʊˈteɪʃn/ | 旋转 | Transform 中的欧拉角旋转值。<br>📌 *"Rotation X: 45 makes it tilt."* |
| 15 | **Scale** | /skeɪl/ | 缩放 | Transform 中的缩放比例。1 为原始大小。<br>📌 *"Scale 2 makes the object twice as big."* |
| 16 | **Asset** | /ˈæset/ | 资源/资产 | 项目中使用的任何文件（模型、贴图、音频、脚本等）。<br>📌 *"Import the 3D model as a new Asset."* |
| 17 | **Prefab** | /ˈpriːfæb/ | 预制体 | 可复用的 GameObject 模板。修改 Prefab 所有实例同步更新。<br>📌 *"Turn the enemy into a Prefab."* |
| 18 | **Material** | /məˈtɪəriəl/ | 材质 | 定义物体表面外观（颜色、纹理、光泽）。<br>📌 *"Apply a red Material to the sphere."* |
| 19 | **Texture** | /ˈtekstʃər/ | 贴图/纹理 | 贴在模型表面的 2D 图片。PNG/JPG 格式。<br>📌 *"The Texture resolution is 1024×1024."* |
| 20 | **Shader** | /ˈʃeɪdər/ | 着色器 | 运行在 GPU 上的小程序，控制渲染效果。<br>📌 *"This Shader creates a water effect."* |
| 21 | **Collider** | /kəˈlaɪdər/ | 碰撞器 | 定义物体的物理碰撞形状。<br>📌 *"Add a Box Collider to detect collisions."* |
| 22 | **Rigidbody** | /rɪˈdʒɪd bɒdi/ | 刚体 | 让物体受物理引擎控制的组件。<br>📌 *"Rigidbody enables gravity and forces."* |
| 23 | **Tag** | /tæɡ/ | 标签 | 用于分类和识别对象的标识符。<br>📌 *"Tag the player as 'Player'."* |
| 24 | **Layer** | /ˈleɪər/ | 层 | 用于碰撞过滤、摄像机渲染分组。<br>📌 *"Put UI elements on the UI Layer."* |
| 25 | **Scene** | /siːn/ | 场景/关卡 | 一个独立的游戏关卡/界面文件 (.unity)。<br>📌 *"Load the next Scene when the level ends."* (注意：Scene 既是视图名也是关卡名) |

| 38 | **Build** | /bɪld/ | 构建/打包 | 将项目编译为可执行文件。<br>📌 *"Build the game for Windows."* |
| 39 | **Player Settings** | /ˈpleɪər ˈsetɪŋz/ | 播放器设置 | 项目构建设置（图标、分辨率、公司名等）。<br>📌 *"Change the company name in Player Settings."* |

# 附 1-A：Unity 菜单栏详解（Menu Bar）

> 菜单栏位于 Unity 编辑器最顶部，按菜单路径逐级列出。

## File（文件）菜单

| 菜单路径 | 中文 | 说明 |
|----------|------|------|
| **File > New Scene** | 新建场景 | 创建一个空白的新场景。<br>📌 *"Start with File > New Scene."* |
| **File > Open Scene** | 打开场景 | 加载已有的 .unity 场景文件。<br>📌 *"File > Open Scene to load the level."* |
| **File > Save** | 保存 | 保存当前场景的修改。<br>📌 *"Remember to Save often."* |
| **File > Save As** | 另存为 | 将场景另存为新文件。<br>📌 *"Save As a new name for backup."* |
| **File > New Project** | 新建项目 | 创建全新的 Unity 项目。<br>📌 *"File > New Project to start fresh."* |
| **File > Open Project** | 打开项目 | 切换到另一个已有的项目。<br>📌 *"File > Open Project to switch projects."* |
| **File > Build Settings** | 构建设置 | 设置目标平台和场景列表。<br>📌 *"Open Build Settings to add scenes."* |
| **File > Build & Run** | 构建并运行 | 打包 + 自动启动测试。<br>📌 *"Build & Run to test on device."* |
| **File > Exit** | 退出 | 关闭 Unity 编辑器。<br>📌 *"File > Exit to close Unity."* |

## Edit（编辑）菜单

| 菜单路径 | 中文 | 说明 |
|----------|------|------|
| **Edit > Undo** | 撤销 | 回退上一步操作 (Ctrl+Z)。<br>📌 *"Edit > Undo to revert."* |
| **Edit > Redo** | 重做 | 恢复被撤销的操作 (Ctrl+Y)。<br>📌 *"Edit > Redo if you undo too far."* |
| **Edit > Cut** | 剪切 | 移除选中项并放入剪贴板。<br>📌 *"Cut the GameObject to move it."* |
| **Edit > Copy** | 复制 | 复制选中项到剪贴板。<br>📌 *"Copy the component to paste later."* |
| **Edit > Paste** | 粘贴 | 从剪贴板粘贴内容。<br>📌 *"Paste the values into the Inspector."* |
| **Edit > Duplicate** | 复制/复制对象 | 直接在原地复制选中对象 (Ctrl+D)。<br>📌 *"Duplicate the enemy to make more."* |
| **Edit > Delete** | 删除 | 移除选中的对象或资源。<br>📌 *"Delete the temporary object."* |
| **Edit > Frame Selected** | 聚焦选中 | 在 Scene 视图中将镜头对准选中对象 (F 键)。<br>📌 *"Press F to Frame Selected."* |
| **Edit > Select All** | 全选 | 选中当前视图中的所有对象。<br>📌 *"Select All to batch-delete."* |
| **Edit > Preferences** | 偏好设置 | 编辑器全局设置（主题、快捷键等）。<br>📌 *"Change the theme in Preferences."* |
| **Edit > Project Settings** | 项目设置 | 当前项目的详细配置（输入、物理、标签等）。<br>📌 *"Open Project Settings to configure input."* |
| **Edit > Shortcuts** | 快捷键 | 查看和自定义快捷键绑定。<br>📌 *"Check Shortcuts to learn key bindings."* |

## Assets（资源）菜单

| 菜单路径 | 中文 | 说明 |
|----------|------|------|
| **Assets > Create** | 创建 | 在项目中创建新资源（文件夹、脚本、材质等）。<br>📌 *"Assets > Create > Folder to organize."* |
| **Assets > Import New Asset** | 导入新资源 | 从外部文件导入资源到项目。<br>📌 *"Import New Asset to bring in a model."* |
| **Assets > Import Package** | 导入包 | 导入 Unity 资源包 (.unitypackage)。<br>📌 *"Import Package for standard assets."* |
| **Assets > Export Package** | 导出包 | 将选中的资源打包为 .unitypackage。<br>📌 *"Export Package to share your assets."* |
| **Assets > Find References In Scene** | 查找场景引用 | 查看某资源在当前场景中被哪些对象使用。<br>📌 *"Find References to trace usage."* |
| **Assets > Reimport** | 重新导入 | 强制重新导入选中的资源。<br>📌 *"Reimport if the asset looks wrong."* |
| **Assets > Refresh** | 刷新 | 刷新资源数据库 (Ctrl+R)。<br>📌 *"Refresh after adding files manually."* |

# 附 1-B：Unity 工具栏 & 视图切换

> 编辑器最上方的快捷按钮区域。

| 单词 | 音标 | 中文 | 说明 |
|------|------|------|------|
| **Toolbar** | /ˈtuːlbɑːr/ | 工具栏 | 编辑器顶部的快捷操作按钮区域。<br>📌 *"The Toolbar has Play and Layout buttons."* |
| **Transform Tools** | /trænsˈfɔːrm tuːlz/ | 变换工具组 | 工具栏左侧的五个变换模式按钮。<br>📌 *"Select the Move tool from Transform Tools."* |
| **Hand Tool** | /hænd tuːl/ | 抓手工具 | 平移 Scene 视图的镜头 (Q 键)。<br>📌 *"Hand Tool to pan the view."* |
| **Move Tool** | /muːv tuːl/ | 移动工具 | 移动选中的对象 (W 键)。<br>📌 *"Move Tool to reposition the object."* |
| **Rotate Tool** | /rəʊˈteɪt tuːl/ | 旋转工具 | 旋转选中的对象 (E 键)。<br>📌 *"Rotate Tool to turn the object."* |
| **Scale Tool** | /skeɪl tuːl/ | 缩放工具 | 缩放选中的对象 (R 键)。<br>📌 *"Scale Tool to resize."* |
| **Rect Tool** | /rekt tuːl/ | 矩形工具 | 同时变换位置/大小/旋转 (T 键)，用于 UI。<br>📌 *"Rect Tool for UI elements."* |
