|------|---------|
| 基础入门 | 变量、函数、指针、结构体 = C++ 的地基 |
| 面向对象 | 类、继承、多态 = 组织代码的方式 |
| 现代C++ | 智能指针、Lambda、并发 = 写安全高效的代码 |
| 工程实践 | CMake、GTest、调试 = 能工作的技能 |

## 遇到问题

1. **看不懂参考文档** → 搜索 `cppreference.com` + 你的问题
2. **编译报错** → C++ 的错误信息很长但很有用，从第一个错误开始修
3. **练习不会做** → 先看参考文档的示例代码，模仿着写
4. **想深入某个话题** → 搜索 `cpp best practices [话题]`
