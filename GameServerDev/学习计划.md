# 游戏服务端开发学习计划

> **目标：** 系统掌握游戏服务端开发，覆盖 C# 服务端编程、Linux 系统编程、数据库、游戏后端逻辑、分布式架构、运维工具
> **方法：** 理论 + 动手交替进行，每个知识点都有可运行的示例
> **前置：** 已有 Unity 客户端开发基础（UnityClientDev）、C++（Raylib 13 个项目）、Lua（Pico-8）基础

## 课程总览

| 阶段 | 天数 | 主题 | 对应后端技能点 |
|------|------|------|---------------|
| 一 | Day01~05 | C# 服务端编程 | 网络编程、多线程、协议、日志 |
| 二 | Day06~10 | Linux 与系统编程 | 操作系统、epoll、Docker |
| 三 | Day11~15 | 数据库 | MySQL、Redis、数据库设计 |
| 四 | Day16~20 | 游戏后端逻辑 | 服务器架构、账号、战斗、排行 |
| 五 | Day21~24 | 分布式与高可用 | CAP、一致性哈希、消息队列、安全 |
| 六 | Day25~27 | 运维与工具开发 | 部署监控、日志分析、GM 工具 |
| 项目 | 实战项目 | 完整游戏服务器 | 综合运用 |

## 学习模式说明

> 每个主题有 **基础版**（`DayXX_主题.md`）和 **深入版**（`DayXX_主题_深入.md`）两个文件。

| 文件类型 | 说明 | 建议时间 |
|---------|------|---------|
| 基础版 | 核心概念 + 基础用法 + 简单示例 | 20-30 分钟 |
| 深入版 | 进阶原理 + 底层机制 + 实战技巧 | 30-45 分钟 |

**推荐学习路径：**
1. **快速入门**：只读基础版 → 完成练习 → 进入下一主题
2. **深入理解**：读完基础版 → 读深入版 → 对比客户端已有知识
3. **查漏补缺**：遇到具体问题时，针对性阅读深入版相关章节

## 时间安排

> 你已有 Unity 客户端基础 + C++/Lua 经验，后端重点放在 **系统编程**、**高性能网络**、**数据库原理** 以及 **分布式架构**。

```
22:00 ─ 22:05  回顾昨天的笔记（5分钟热身）
22:05 ─ 22:25  读概念 + 看示例代码（20分钟）
22:25 ─ 22:40  动手改代码 / 写小练习（15分钟）
22:40 ─ 22:45  记录一条你今天学到的东西（停靠点）
22:45         停！关电脑，让大脑离线
```
